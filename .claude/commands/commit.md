コミットメッセージを生成して、コミットします。

## 実行条件

- `git status` で変更があることを確認してください。

## 実行プロセス

### フェーズ1: 差分の確認

- `git diff` で差分を確認してください。

### フェーズ2: コミットメッセージの生成

- `git commit -m "コミットメッセージ"` でコミットしてください。
- コミットメッセージは1文以内かつ簡素に記載してください。
- コミットメッセージは英語で記載してください。
- コミットメッセージに`generated by Claude`のような署名は含めないでください。
